# Changelog

## [v2.0.0] - 2026-05-30

### Added

- Added Schema v2.0 for structured battery safety standard data.
- Added 21 structured battery safety standard JSON files.
- Added taxonomy for CELL / MODULE / PACK / SYSTEM levels.
- Added ELEC / MECH / THERM test classification.
- Added application-domain navigation matrix.
- Added horizontal comparison grid.
- Added AI Consultant interface through Vercel Edge Function.
- Added automated audit, crawler, and standard version checking scripts.

### Changed

- Migrated frontend architecture to React 18 + Vite.
- Improved UI for standard comparison and missing-test-condition prompts.
- Improved standard version visibility and outdated-standard warnings.
- Reorganized project structure for data, scripts, docs, API, and frontend.

### Maintenance

- Established this release as the first formal versioned baseline.
- Future updates will track schema changes, standard updates, data corrections, and audit results through Issues and Releases.
