## 3. 判定標準：危害嚴重程度等級 (Hazard Severity Level, HSL)

本標準使用 HSL 0-7 級進行判定，記錄測試過程中的最高等級。

| 等級 | 描述 | 判定標準細節 |
| --- | --- | --- |
| 0 | 無影響 | 無功能損失。 |
| 1 | 被動保護啟動 | 保護裝置啟動（如保險絲），更換後可恢復功能。 |
| 2 | 缺陷/損壞 | 無危害但 RESS 損壞，不可逆的功能損失。 |
| 3 | 輕微洩漏/排氣 | 有排氣跡象，但無明顯液體積聚或煙霧阻塞視線。 |
| 4 | 嚴重洩漏/排氣 | 明顯液體積聚，煙霧阻塞視線，質量損失 < 30%。 |
| 5 | 破裂 (Rupture) | 外殼機械完整性喪失，內容物釋放，質量損失 30-55%。 |
| 6 | 起火 (Fire) | 持續燃燒 ≥ 1 秒。 |
| 7 | 劇烈失效 | 能量快速釋放，產生壓力波或碎片，質量損失 ≥ 55%。 |

## 4. 豁免與特殊情況 (Exemptions and Special Cases)

*   **電化學電容器 (Capacitors)**：
    *   **IF**：受測設備為電容器而非化學電池。
    *   **THEN**：執行「過電壓測試」(Overvoltage) 與「電壓反轉測試」(Voltage Reversal) 代替過充/過放測試。
*   **熱升溫測試限制**：電池包 (Pack) 等級通常不建議進行整包熱升溫測試。
*   **過充電流測試豁免**：若充電系統的最大能力低於 RESS 的最大容許電流，可豁免該測試。

## 5. 操作邏輯與流程 (Operational Logic and Workflow)

### 5.1 測試終止邏輯 (End Conditions)
*   **過充電測試**：達到以下任一條件即終止：
    1.  發生故障 (HSL ≥ 5)。
    2.  達到 250% SOC。
*   **外部短絡測試**：
    *   持續至少 60 分鐘，或直到發生故障 (HSL ≥ 5)。
*   **壓壞測試停止邏輯**：
    1.  發生故障 (HSL ≥ 5)。
    2.  達到 100% 實際位移。
    3.  壓力達到上限值 (如 25 kN 或模組級 500 kN)。

### 5.2 穿刺測試矩陣邏輯 (Penetration Logic)
若樣品充足，建議從最嚴苛條件開始 (100% SOC / 55°C)。
*   **IF** 該條件下 HSL 評級可接受：**THEN** 繼續測試較低 SOC 或溫度條件。
*   **強制位移停止**：電壓降 ≥ 100 mV 或發生故障時停止。
