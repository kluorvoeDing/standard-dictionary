## 3. 判定標準 (Pass/Fail Criteria)

### 3.1 核心判定準則
*   **不發火 (No fire)**：測試過程中及測試後觀察期內不得出現明火。
*   **不爆炸 (No explosion)**：測試過程中不得出現劇烈破裂或噴發。
*   **外殼完整性 (Case Integrity)**：對於蔓延測試，電池系統外殼不得破裂 (No case rupture)。
*   **BMS 功能 (BMS Functionality)**：BMS 必須在超過電芯上限充電電壓前終止充電。

## 4. 豁免與特殊情況 (Exemptions and Special Cases)

*   **過充電測試豁免 (Overcharge Test Waiver)**：
    *   **IF**：電池系統提供兩個或更多獨立的充電電壓控制保護（例如：電芯電壓監測 + 總電壓診斷監測）。
    *   **THEN**：該測試可以豁免。
*   **過充電電流控制測試豁免 (Overcharge Current Control Waiver)**：
    *   **IF**：上層系統的最大充電電流能力低於電池系統的最大容許充電電流。
    *   **THEN**：該測試可以豁免。
*   **蔓延測試觀察 (Propagation Observation)**：
    *   誘發用的目標電芯發火或外殼破裂是可接受的，但不得蔓延至整個系統。

## 5. 操作邏輯與流程 (Operational Logic and Workflow)

### 5.1 跌落測試邏輯 (Drop Test Selection Logic)
*   **IF** 質量 < 7 kg：
    *   執行「全跌落」(Whole Drop)，高度 1000 mm，隨機方向 3 次。
*   **IF** 7 kg ≤ 質量 < 20 kg：
    *   執行「全跌落」(Whole Drop)，高度 100 mm，指定底部朝下 3 次。
*   **IF** 質量 ≥ 20 kg：
    *   執行「棱角跌落」(Edge and Corner Drop)，高度依重量等級 (25/50/100 mm)，棱角各 2 次。

### 5.2 強制放電目標電壓邏輯 (Forced Discharge Target Voltage)
*   **情況 A**：系統具有兩級獨立電壓保護或僅單電芯。
    *   目標電壓 = 負的電芯上限充電電壓。
*   **情況 B**：系統僅有單級或無電壓保護。
    *   目標電壓 = -(n-1) × 電芯上限充電電壓 (n 為串聯電芯數)。

### 5.3 強制內部短絡停止邏輯 (Internal Short Pressing Logic)
*   **IF** 檢測到電壓降 > 50 mV：**立即停止**下降並保持 30s。
*   **IF** 壓力達到上限 (圓柱 800N / 方形 400N)：**立即停止**下降並釋放。
